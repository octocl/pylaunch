# Future Features

## Short-term (post-MVP)

- [ ] OAuth login (GitHub, Google)
- [ ] Python version selector (3.9, 3.10, 3.11, 3.12)
- [ ] Shareable project links (read-only view + run)
- [ ] Premium subscription via Stripe
- [ ] Priority execution queue for premium users
- [ ] Environment variable management per project
- [ ] Light / dark mode toggle
- [ ] File download from container (premium)

## Medium-term

- [ ] REST API keys for programmatic execution
- [ ] Webhook triggers (run script when an event fires)
- [ ] Scheduled execution (cron-like, premium)
- [ ] Team / organization accounts
- [ ] Usage dashboard (execution count, total runtime, etc.)
- [ ] Email notifications (execution complete, project shared)

## Long-term

- [ ] Plugin system (pre-installed libraries per project)
- [ ] Community script gallery (user-submitted, curated)
- [ ] Fork / remix other users' public projects
- [ ] Collaboration features (multi-user editing)
- [ ] GPU-backed execution for ML (premium add-on)
- [ ] Support for additional languages (Node.js, Go, Rust)
- [ ] Native mobile apps (iOS/Android)
- [ ] On-premise deployment option for enterprises

## Deliberately not planned

- Shell/terminal access to containers
- Persistent background processes
- File storage beyond project code
- Package registry or deployment service
