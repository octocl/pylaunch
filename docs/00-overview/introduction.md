# Introduction

PyLaunch is a free, ad-supported cloud platform that lets anyone run Python scripts inside isolated Docker containers — no local Python installation required.

Upload a `.py` file or write code in the browser editor, hit run, and see the output in real time. Every execution spins up a fresh Docker container with internet access, so scripts can install packages, call APIs, and do everything a normal Python environment can.

## Who is it for?

- Beginners learning Python who don't want to set up a local environment
- Developers who need to quickly test a script without `pip install`
- Hobbyists running Telegram/Discord bots, web scrapers, or automation
- Students submitting programming assignments
- Anyone who wants to run Python from a phone, tablet, or Chromebook

## Core principles

- **Zero setup** — open a browser and start coding
- **Free forever** — supported by non-intrusive ads
- **Isolated** — every run is a fresh container, no state leaks
- **Internet-connected** — scripts can reach external services

## What it is not

PyLaunch is not a hosted IDE, not a production deployment platform, and not a replacement for a full VPS. It is a lightweight, low-friction way to run Python code on demand.
