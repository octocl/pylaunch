# Unique Selling Points

PyLaunch differentiates itself by focusing on what matters to its users — speed, simplicity, and zero friction — rather than feature-count comparisons with competitors.

| USP | Details |
|---|---|
| **Free forever** | No credit card required. No trial period. Free tier is sustainable via ads. |
| **Zero install** | Works in any browser. No Python, no pip, no virtualenv. |
| **Instant execution** | Container boots in under 2 seconds for common images. |
| **Docker isolation** | Every run is a fresh container — no state leakage between runs. |
| **Internet access** | Scripts can `pip install`, call REST APIs, connect to databases. |
| **Project persistence** | Registered users can save, version, and re-run projects. |
| **Live logs** | stdout/stderr streams to the browser in real time via WebSocket / xterm.js. |
| **Modern UI** | Monaco editor, syntax highlighting, dark mode, responsive layout. |
| **No account required** | Guest users can upload and run immediately (ad-supported). |
| **Beginner friendly** | Designed for people who have never touched a terminal. |

## What we don't do

- No hidden costs or surprise billing
- No terminal/SSH access to containers
- No persistent filesystem (except saved projects for registered users)
- No long-running processes (executions have a time limit)
