# Registered Users

Creating an account unlocks persistence and history features while keeping the same core execution experience.

## Account creation

- Email + password (min 8 characters)
- OAuth options (GitHub, Google) — future
- Email verification (optional for MVP, required for sensitive features)

## Features unlocked

| Feature | Details |
|---|---|
| **Project storage** | Create, rename, edit, delete projects |
| **Auto-save** | Code is saved on every run and periodically while editing |
| **Execution history** | Full log of every run with output, exit code, duration |
| **Re-run** | One-click re-run of any previous execution |
| **Project sharing** | Optional public share link for each project |
| **Ad-supported** | Free registered users see the same ads as guests |

## Project limits (free tier)

| Limit | Value |
|---|---|
| Max projects | 50 |
| Max code size | 100 KB |
| History retention | 30 days |
| Share links | Yes (public) |

## Dashboard

The dashboard is the home base for registered users:

- List of projects with last-run timestamp
- Quick search / filter
- Execution history timeline
- One-click "New project" and "Run"
