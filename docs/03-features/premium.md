# Premium

The premium tier is a monthly subscription that removes limitations and provides a better experience.

## Pricing model

- Monthly subscription (e.g., $5/month or local equivalent)
- No annual commitment required
- No ads — ever

## What premium unlocks

| Feature | Free | Premium |
|---|---|---|
| Ads | Yes | No |
| Execution timeout | 120s | 300s |
| Memory limit | 256 MB | 1 GB |
| CPU limit | 0.5 cores | 1 core |
| Concurrent runs | 1 | 3 |
| Max projects | 50 | Unlimited |
| History retention | 30 days | 1 year |
| Env variables | No | Yes |
| Priority queue | No | Yes |

## Payment integration

- Processed via Stripe or similar
- No credit card storage on our servers
- Invoicing available on request

## Subscription management

Users can manage their subscription from the account settings page:

- View current plan
- Upgrade from free to premium
- Cancel subscription (access continues until end of billing period)
- View payment history
