# Homepage

The landing page serves two purposes: explain what PyLaunch is, and get users to try it immediately.

## Layout

```
┌──────────────────────────────────────────────────┐
│  Logo                              [Sign In]      │
├──────────────────────────────────────────────────┤
│                                                    │
│  ┌──────────────────────────────────────────────┐ │
│  │  Run Python in your browser.                 │ │
│  │  Free. No install.                           │ │
│  │                                              │ │
│  │  [Try it now]  [Learn more]                  │ │
│  └──────────────────────────────────────────────┘ │
│                                                    │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐     │
│  │ No     │ │ Docker │ │ Live   │ │ Free   │     │
│  │ Install│ │ Secure │ │ Output │ │ Forever │     │
│  └────────┘ └────────┘ └────────┘ └────────┘     │
│                                                    │
│  ┌──────────────────────────────────────────────┐ │
│  │ [Screenshot of editor with code running]     │ │
│  └──────────────────────────────────────────────┘ │
│                                                    │
│  - How it works (3 steps: Write, Run, See)        │
│  - Use cases (bots, scraping, learning)            │
│  - Pricing comparison table                        │
│  - Footer                                          │
└──────────────────────────────────────────────────┘
```

## Key elements

| Element | Description |
|---|---|
| Hero headline | Clear value proposition in one sentence |
| CTA buttons | "Try it now" goes to editor; "Learn more" scrolls to features |
| Feature icons | Four key benefits at a glance |
| Editor preview | Interactive-looking screenshot or live demo |
| How it works | 3-step explainer (Write → Run → See) |
| Use cases | Quick cards showing real applications |
| Pricing | Free vs Premium comparison |
| Footer | Links, legal, social |
